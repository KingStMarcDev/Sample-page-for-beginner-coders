📱 Sample Page for Beginner Coders
A clean, responsive mobile-style web app with a bottom navigation toolbar. Perfect for learning how to build multi-page interfaces using HTML, CSS, and vanilla JavaScript.

🎯 What you'll learn:
Fixed bottom navigation bar (toolbar)

Dynamic content switching without page reload

Active state highlighting for current tab

Card-based UI design

Image and content placeholders

CSS transitions and hover effects

🧩 Features:
4 fully functional tabs: Home, Add, Market, Settings

Modern UI: Glassmorphism toolbar with blur effect

Responsive layout: Works on desktop and mobile devices

Easy to customize: Change colors, icons, text, and images in one place

No dependencies: Pure HTML/CSS/JS — no frameworks or libraries

🛠️ How to use:
Clone or download the HTML file

Open it in any modern browser

Click the bottom icons to switch between pages

Edit the showPage() function to add your own content

Modify CSS variables to match your brand colors

🎨 Customization ideas:
Replace placeholder images with your own URLs

Add more tabs by copying button elements

Change the active color (#1d4ed8) to any hex value

Modify toolbar background and transparency

Add forms, lists, or interactive elements inside cards

🌟 Perfect for:
Beginners learning frontend development

Students building portfolio projects

Prototyping mobile app interfaces
